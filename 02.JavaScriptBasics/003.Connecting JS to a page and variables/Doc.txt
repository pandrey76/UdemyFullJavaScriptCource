    Для начала работы мы переходим в файл index.html, запускаем 
live server (Кликаем Go live в статусной строке)  и открываем консоль разработчика 
(Меню браузера->Дополнительные инструменты->Инструменты разработчика).

Вкладки Консоли разработчика (на примере браузера Chrome)
  Elements - здесь мы видим описание нашей вёрстки, здесь видны все теги
              какие у нас есть, применяемые к ним стили и т.д., здесь
              даже можно посмотреть обработчики событий (Event Listner);
  Console - это основная наша вкладка, где мы будем работать. Сюда 
            выводятся различные ошибки и предупреждения или даже 
            сообщения от разработчика. К тому же консоль всегда показывает,
            где у вас была ошибка, в какой строчке, в каком файле и полное 
            описание этой ошибки.
  Sources - здесь мы можем найти все материалы и приложения, которые 
            доступны к нашему сайту, именно которые нам доступны, потому что
            многие из них могут быть скрыты.
  Network - где мы сможем отлеживать какие то запросы и т.п. вещи.
  Performance - тесты производительности нашей страницы.
  Memory - отслеживание сколько наша страница занимает памяти.
  Application - здесь расположены различные утилиты типа: Cookies, Web 
                storage, Local storage и т.д.
  Security - информация по шифрованию страницы.
  Остальные вкладки добавляются при помощи плагинов.

    Обязательно комментируйте код!!
    Для JS комментарии такиеже как для c++:
      // - однострочный;
      /* ... */ - многострочный.
    В VScode многострочные комментарии делаются так, выделяется кусок кода,
а потом нажимается комбинация клавиш 'Ctrl+/'.
