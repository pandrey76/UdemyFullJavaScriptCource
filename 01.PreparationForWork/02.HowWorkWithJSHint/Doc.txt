JSHint необходимо устанавливать через npm:
Для linux : sudo npm install -g jshint

Настройки JSHint:
{
    "camelcase" : true, // Форма camelcase
	"indent": 2,        // Настройки отступов
	"undef": true,      // Настройки переменных
	"quotmark": false,  // Настройки кавычек, чтоьы мы могли писать 
                           только одинарные, только двойные
                            или это не важно!
	"maxlen": 80,       // Максимальная длинна строки (80 символов),
                           если он будет больше то среда начнёт ругаться.
	"trailing": true,
	"curly": true,      // Обязательно лт здесь фигурные ковычки.
	"strict": false,    // Использование строгого режима
	"browser": true,    // Использование браузера
	"devel": true,
	"jquery": true,
	"esversion": 6
}
        В будущем мы уже будем настраивать свой конфиг с теми настройками, 
    которые мы посчитаем нужными для работе.

        JSHint часто даже при правильном написании кода, всё равно 
    ругается, здесь главное, чтобы это были Warnings, а не Errors.

        Для форматирования кода мы делаем следующее: выделяем, вызываем 
	контекстное меню и жмем опцию 'Форматирование документа' или 
	комбинацию клавиш 'Alt+Shift+F', так работает плагин 'Beautify'.
